# Tugas Eksplorasi: Binary Representation


```
Operator : &
Deskripsi : Membandingkan 2 bit dan mengembalikan nilai 1 jika keduanya  1 bit jika tidak maka 0
Contoh : 1 & 1 = 1

Operator : |
Deskripsi : Membandingkan 2 bit dan mengembalikan nilai 1 jika salah satu bernilai 1 bit jika tidak ada maka 0
Contoh : 1 | 0 = 1

Operator : ^
Deskripsi : Membandingkan 2 bit dan mengembalikan nilai 1 jika kedua bit berbeda dan 0 jika sama
Contoh : 1 ^ 0 = 1;1 ^ 1 = 0

Operator : <<
Deskripsi : Menggeser ke kiri operan pertama, operan ke 2 menentukan banyaknya tempat yang bergerser
Contoh : 5 << 1 = 10

Operator : >>
Deskripsi : Menggeser ke kanan operan pertama, operan ke 2 menentukan banyaknya tempat yang bergerser
Contoh : 1 >> 5 = 0
```
